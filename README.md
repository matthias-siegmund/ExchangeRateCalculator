# ExchangeRateCalculator
This app was built as an example on how to fetch data from an api in an interval of 1 second and display it in a list. It was built in a multi modular approach based on features.

## Techstack
* `MVVM`
* `Kotlin`
* `AndroidX`
* `Dagger for Android`
* `Retrofit 2`
* `Moshi`
* `OkHttp 3`
* `RxJava 2`
* `Timber`

## Modules

* `app`
* `core`
* `exchange`
